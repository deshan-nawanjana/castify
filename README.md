# Movie Streaming with Google Cast API

Castify is a movie library application with built-in support for streaming films to smart devices. Using the Google Cast API, users can effortlessly cast movies from the library to smart TVs and other compatible devices, providing a smooth and convenient viewing experience.

Launch Now: [https://deshan-nawanjana.github.io/castify/](https://deshan-nawanjana.github.io/castify/)

[Read the full technical article](https://deshan.lk/blog/stream-videos-using-google-cast-api)

## Key Features

- Cast video into smart devices such as smart TVs and other compatible devices.
- Load additional content, including poster, title, description, and closed captions.
- Control video playback such as time, play pause state, and captions switch.

## Libraries and Frameworks Used

- [Google Cast API](https://developers.google.com/cast/docs/reference) - Allows developers to enable apps to stream content
- [Vue.js](https://vuejs.org/) - Progressive framework for building user interfaces
